## Project overview

This is a `copier` template for a `devcontainer` with `claude` to allow users to safely run `claude` with restricted system and network access.

## Development guidelines

- Default branch to start from is `master`
- Only one feature/bug should be worked on at a time
- Each feature should get a new branch
- Follow existing code style. Check neighboring files for patterns
- Never co-author commits
- Use "conventional commit" style

## GitHub

- Interact with GitHub using the CLI `gh`
- Prepend all issue bodies with `Authored by Claude 🤖`
