# Claude Code Devcontainer Sanbox

This repo provides a scaffold for setting up a project with a devcontainer that allows the user to run Claude Code more safely (in particular with `--dangerously-skip-permissions`). 

The devcontainer code and docker image are based on the [official Anthropic example](https://github.com/anthropics/claude-code/tree/main/.devcontainer). ([Official docs](https://code.claude.com/docs/en/devcontainer)). 

It is mainly intended to be used in conjunction with VSCode via the official devcontainer extension.
